# Photo Effects - VERSIONING.md

## Application Version

Format

Photo Effects vMajor.Minor.Patch

Example

v0.3.1

## Feature Version

Each feature evolves independently.

Example

Mirror .......... v1.0.0 Crystal Ball .... v0.1.0

## Patch Rules

Patch: - Bug fixes - UI tweaks

Minor: - New capability - Backward compatible

Major: - Architecture changes - Breaking changes

## Release Types

Development - v0.x.x

Beta - v0.9.x

Production - v1.x.x
