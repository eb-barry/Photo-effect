# Photo Effects - UI_GUIDELINE.md

Version: 1.0

## Design Language

Inspired by Apple's Human Interface Guidelines with a Tiffany Green
visual identity.

## Color Palette

Primary: - Tiffany Green (#0ABAB5)

Supporting: - White - Dark Teal - Light Glass White

## Icons

### Feature Icons

-   Glass style
-   Soft 3D
-   Rounded corners
-   Consistent lighting
-   WebP format

### UI Icons

-   White line icons
-   Transparent PNG
-   No background
-   Consistent stroke width

## Typography

-   Clean sans-serif
-   High contrast
-   Three font sizes:
    -   Small
    -   Medium
    -   Large

## Layout

Home Screen - 4 columns (normal) - 3 columns (large font) - Up to 28
functions

Navigation - Home: top-left - Settings: home screen top-right - Open /
Save / Share: top-right in editor pages

## Controls

Buttons - Large touch targets (minimum 48px) - Rounded corners - Glass
appearance where appropriate

Sliders - One slider only per feature parameter - Parameter selected by
dropdown - Selected item visually highlighted

## Accessibility

-   Senior-friendly
-   High contrast
-   Large icons
-   Large text
-   Minimal wording
-   Consistent interaction

## Responsive

Support: - Android phones - iPhone - Tablets - Desktop browsers

## Animation

Use subtle animations only. Avoid distracting effects. Prioritize
performance.
